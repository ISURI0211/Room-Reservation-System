Help & User Guide

This page helps new staff members understand how to use the reservation system.

1) Login

Use your staff username and password to log in. If login fails, check credentials or contact admin.

2) Add Reservation

Enter reservation details carefully. Reservation number must be unique. Dates must be valid (check-out after check-in).

3) View Reservation

Search by reservation number to view full details.

4) Billing

Generate the bill using reservation number. Total is calculated as nights × room rate.

5) Reports

Use daily report to view reservations for a date and revenue report to calculate income for a range.

6) Logout

Always logout when leaving the computer to keep the system secure.

Security Reminder

Do not share login tokens. Logout clears session data and protects guest information.

This system uses token-based sessions stored in browser sessionStorage.
